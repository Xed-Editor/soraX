# soraX

**soraX** is a modified version of **sora-editor**, intended **only** for use within **Xed-Editor**.

Changes from upstream may be applied directly as needed, rather than waiting for official releases from the upstream **sora-editor** project.

This repository is **not** intended for general use.
We do **not** accept bug reports and only **rarely** accept pull requests, unless they are specifically related to **soraX** or **Xed-Editor**.
